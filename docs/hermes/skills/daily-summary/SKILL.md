# Daily Summary Skill

每日整理用户和戏命师的上下文，形成长期记忆。

## 输入

- 当天或最近 20 条对话。
- 用户已有 USER.md / MEMORY.md。
- Dreamer 的 agentState、entitlement、命盘摘要。

## 输出

写入用户私有记忆，不写入共享核心文档。

建议格式：

```markdown
## YYYY-MM-DD

### 稳定事实
- ...

### 用户偏好
- ...

### 长期目标
- ...

### 待跟进事项
- ...
```

## 禁止保存

- 密码、私钥、支付信息、身份证件、服务器凭据。
- 其他用户的隐私。
- 平台后台配置和模型 key。
