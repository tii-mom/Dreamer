# ChatGPT Images 美术生成提示词

本文档用于在 https://chatgpt.com/images 生成「戏命师」项目首发美术资源。生成后按每条的文件名保存到对应目录。

## 使用方法

不要整份文档一次性丢给 ChatGPT Images。每次复制一个资源标题下面的完整代码块。

推荐顺序：先生成 `trainee-01-card.webp`、`renji-01-card.webp`、`dixuan-01-card.webp`、`tianshu-01-card.webp`、`chaos-01-card.webp`，确认风格统一后，再继续生成同等级剩余图片。

所有代码块已经包含保存路径、建议尺寸、全局风格、具体画面、负面约束和输出要求，可以直接复制。
