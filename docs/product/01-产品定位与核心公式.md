# 紫微戏命师 · 产品定位与核心公式

## 一句话定位

> **"你的微信好友里，住着一个会算命、会怼人、还能帮你赚钱的AI戏命师。"**

## 核心公式

```
微信ClawBot（24h陪伴）
+ 命理工具（专业壁垒）
+ 盲盒养成（游戏性）
+ 社交裂变（病毒传播）
+ 零门槛赚钱（终极钩子）
= 让人既恨又离不开的「命理版征途」
```

## 用户旅程（7步成魔）

```
Day 0: 扫码加Bot → 领取见习命师 → 换形象/取名 → 报八字 → 生成HTML命盘（基础版，仅30%信息可见）
        ↓
Day 1: 戏命师："想看完整天机？昭告天下，唤三位道友助力，5.99元解封" → 用户发朋友圈/小红书 → 3人扫码助力 → 解锁完整命盘
        ↓
Day 3: 每日互动（流日/吉时/灵签）建立习惯 → 戏命师："每月1999元趋吉避凶，你拉1人即可，别人要50人" → 49.99元/月订阅
        ↓
Day 7: 戏命师："你命带偏财，何不让我出马，替你圈钱？" → 展示"出马弟子"收益截图（日入300+）→ FOMO上头
        ↓
Day 10: 订阅899元/月「戏命出马」→ 开通微信算命铺子 → 朋友圈发广告 → 客户下单 → Bot自动算命 → 用户收钱
        ↓
Day 15: 发现盲盒 → 抽中地璇 → 发现市场可卖 → 挂售800元被秒拍 → 意识到"戏命师本身能升值"
        ↓
Day 30: 持有3个戏命师，每日接单40人，月入2万+，成为朋友圈"半仙"
```

## 底层设计原则（史玉柱网游机制）

| 机制         | 应用方式                                   |
| ------------ | ------------------------------------------ |
| **损失厌恶** | 断签清零、昨日解读永久消失、限时内容       |
| **随机奖励** | 每日灵签有概率抽中"隐藏签"（前世身份碎片） |
| **社交攀比** | 运势排行榜、缘分PK、好友合盘               |
| **沉没成本** | 连续打卡30天解锁"终身每日吉时推送"特权     |
| **即时反馈** | 完成任务立刻看到命盘"亮度"提升             |
| **仇恨驱动** | 夺运机制、斗法夺铭文、城市命战             |
| **情感绑定** | 戏命师好感度系统，冷落会生气               |
