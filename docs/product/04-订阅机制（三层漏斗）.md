# 紫微戏命师 · 订阅机制（三层漏斗）

## 第一层：命盘解封（一次性）

- **标价**：599元
- **戏命师话术**：
  > _"既然你今日将我封印打开，只要你将此消息昭告天下，且有三位好友助力，仅需5.99元我将天机泄漏于你。记住，是泄漏，不是泄露——天机太重，漏一点就够了。"_
- **动作**：生成带二维码的"命盘封印图" → 发朋友圈/小红书 → 3人扫码助力 → 支付5.99元 → 解锁完整HTML命盘 + 大运流年 + 四化飞星

## 第二层：趋吉避凶（月订阅）

- **标价**：1999元/月
- **戏命师话术**：
  > _"别人要拉50个蠢货才能凑够气运，你？你拉1个就够了。我观你面相，不是做拼多多的命。49.99元，我每日陪你趋吉避凶，可好？"_
- **权益**：
  - 每日流日盘解读（HTML推送）
  - 每日吉时/凶时提醒
  - 每日问事3次（事业/感情/出行）
  - 每日灵签（可生成分享卡片）
  - 突发星象预警（微信主动推送）
  - 每月1次八字精批

## 第三层：戏命出马（赚钱权限，月订阅）

- **标价**：12999元/月
- **实际价**：899元/月（限时，戏命师："这价格我只说一次，明日卯时恢复原价"）
- **权益**：
  - 开通「算命铺子」（微信小店/小程序）
  - 每日额度：10人紫微斗数 + 10人星盘 + 20人运势占卜
  - 用户自主定价：10-100元/人
  - 平台仅收token成本费（约0.5-2元/单，按模型消耗实算）
  - 戏命师自动接单、自动排盘、自动生成解读报告
  - 用户只需：发朋友圈获客 → 收钱

### FOMO设计

- 每日展示「昨日出马弟子收益榜」：_"昨日'深圳张三'靠我入账1,847元，你还在等什么？"_
- 限时：每月只开放500个「戏命出马」名额（稀缺性）
- 退款承诺：7天内无订单，全额退（降低决策门槛）
