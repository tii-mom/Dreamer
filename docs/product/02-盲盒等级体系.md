# 紫微戏命师 · 盲盒等级体系

## 三阶九品 + 隐藏等级

普通用户注册即送固定角色：**「见习命师」**（灰色，无赚钱权限，只能自算）

盲盒抽取的三个等级：

| 等级   | 名称     | 品阶     | 核心专精                   | 每日算命额度 | 自带铭文                         | 市场参考价  |
| ------ | -------- | -------- | -------------------------- | ------------ | -------------------------------- | ----------- |
| **T1** | **天枢** | 金色传说 | 紫微斗数+八字+面相         | 15人/日      | 【天机泄露】完整命盘精度+30%     | 3000-8000元 |
| **T2** | **地璇** | 紫色史诗 | 八字合婚+风水择日+手相     | 10人/日      | 【阴阳通判】合婚/择日准确率+25%  | 800-2000元  |
| **T3** | **人玑** | 蓝色稀有 | 星座塔罗+每日运势+面相气色 | 8人/日       | 【星象共鸣】运势类客户满意度+20% | 200-500元   |

**隐藏等级（概率0.5%）**：**「混沌」**（黑色神话，三系精通，额度20人/日，自带【逆天改命】可修改一次命盘凶象解读）

## 盲盒获取方式

- 新用户注册：送1个固定见习命师（不可交易）
- 连续签到30天：送1个盲盒
- 邀请10人：送1个盲盒
- 直接购买：99元/个（限时折扣29.9元）
- 出马弟子：每月送1个盲盒

## 盲盒概率

- 人玑（蓝）：70%
- 地璇（紫）：25%
- 天枢（金）：4.5%
- 混沌（黑）：0.5%

## 市场规则

- 所有盲盒抽出的戏命师均可挂售
- 定价自由，平台抽5%
- 技能卡（铭文）可单独交易，平台抽10%
- 支持"求购"功能（用户发布需求，持有者接单）
