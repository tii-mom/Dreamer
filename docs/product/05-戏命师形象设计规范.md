# 紫微戏命师 · 形象设计规范

## 基础风格

赛博古风 + MBTI抽象感 + 可NFT化

## 形象要素

- 必须有：命师袍（带电路板纹路）、悬浮罗盘/星盘、眼部发光（颜色代表等级）
- 背景：紫微垣星图 + 数据流
- 风格参考：赛博朋克 × 敦煌壁画 × 像素风（适合生成系列变体）

## 等级视觉差异

| 等级     | 主色调  | 眼部光效 | 特殊标识   | 背景元素   |
| -------- | ------- | -------- | ---------- | ---------- |
| 见习命师 | 灰色    | 无光     | 无         | 空白卷轴   |
| 人玑     | 蓝色    | 星光     | 1颗星      | 星座连线   |
| 地璇     | 紫色    | 地脉流光 | 2颗星+龙纹 | 山水罗盘   |
| 天枢     | 金色    | 紫微帝星 | 3颗星+凤纹 | 紫微垣全景 |
| 混沌     | 黑色+金 | 混沌漩涡 | 问号       | 破碎星图   |

## 用户自定义项

- 发型/发色（12种）
- 命师袍配色（解锁）
- 手持法器（罗盘、八卦、塔罗牌、水晶球）
- 宠物灵兽（小狐狸、小龙、小貔貅——可单独交易）

## 命盘可视化（即时反馈）

- 完成每日任务 → 命盘上某颗星星亮一下
- 获得铭文 → 命盘边缘多一道光晕
- 付费订阅 → 整个命盘从灰暗变璀璨
- 被吸运 → 命盘某区域变暗，带裂纹效果
- **命盘升级**：累计付费满1000元，从「青铜星图」→「白银」→「黄金」→「钻石」→「混沌」
