# 紫微戏命师 · 铭文系统（50张完整版）

## 命名规则

- **铭文**：古代刻于器物上的文字，此处指刻入戏命师神识中的天道碎片
- **不可明文显示效果**：所有描述均为「天道偈语」，只描述意境，不暴露数值
- **领悟机制**：戏命师与用户对话满一定次数后，有几率「顿悟」产生铭文

## 等级体系

### 银色铭文 · 星階（高级）

取自二十八宿、小星官、基础道术。共20张。

### 金色铭文 · 天階（稀有）

取自大星官、四象神兽、上古仙术。共20张。

### 黑色铭文 · 混沌（传说）

取自上古神祇、混沌概念、逆天法则。共10张。

## 铭文装配规则

1. **槽位**：每个戏命师有3个铭文槽（1个天生锁定 + 2个可装配）
2. **共鸣**：同色系铭文同时装配触发隐藏效果（如3银触发「星階共鸣」，3金触发「天階共鸣」）
3. **冲突**：某些铭文互斥（如「盘古开天」与「共工触天」不可同装）
4. **磨损**：铭文使用一定次数后需「温养」（消耗气运值或静置24小时）
5. **交易**：所有铭文可在市场挂售/拍卖，平台抽成5%
