# 紫微戏命师 · 赚钱模式（戏命出马）闭环

## 用户操作流

1. 订阅899元 → Bot自动开通「微信算命铺子」
2. 用户设置定价（建议30-50元/单）
3. 用户发朋友圈广告（Bot提供每日不同文案）
4. 客户点击链接 → 进入微信小店/小程序 → 选择服务类型 → 支付
5. Bot自动接单 → 客户提交八字/问题 → Bot排盘（iztro/lunar）→ LLM解读 → 生成HTML报告
6. 用户微信收款（T+1到账）
7. 平台自动扣除token费（成本约0.5-2元/单）

## 用户收益模型

| 定价  | 日单量 | 日收入 | 日成本(token) | 日净利 | 月收入   |
| ----- | ------ | ------ | ------------- | ------ | -------- |
| 30元  | 20单   | 600元  | 20元          | 580元  | 17,400元 |
| 50元  | 10单   | 500元  | 10元          | 490元  | 14,700元 |
| 100元 | 5单    | 500元  | 5元           | 495元  | 14,850元 |

## 平台收益

- 订阅费：899元/人/月
- token费：约0.5-2元/单（覆盖AI成本）
- 技能卡市场：抽成10%
- 戏命师交易：抽成5%

## 裂变设计

- 客户算命后，生成"由XXX的专属戏命师测算"水印图
- 客户晒图 → 其好友扫码 → 新用户转化
- 每带来1个新出马弟子，上级获得100元返佣 + 1张技能卡
